# 🎬 Movie Explorer

Movie Explorer is a modern React movie application built using TMDB API.  
Users can browse popular movies, search movies in real time, view detailed movie information, and save favorite movies using local storage.

---

# 🚀 Features

- 🔥 Browse Popular Movies
- 🔎 Search Movies with Debouncing
- 🎞 View Detailed Movie Information
- ❤️ Add / Remove Favorite Movies
- 💾 Favorites Saved in Local Storage
- ♾ Infinite Scroll Pagination
- 📱 Fully Responsive UI
- ⚡ Fast Performance with React + Vite
- 🎨 Modern UI using Tailwind CSS

---

# 🛠 Tech Stack

- React.js
- React Router DOM
- Tailwind CSS
- Axios
- TMDB API
- Vite
