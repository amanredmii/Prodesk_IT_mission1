Wizard Form Project

A modern multi-step Wizard Form built using React and Tailwind CSS.  
This project provides a clean and interactive user experience with step-by-step form navigation, validation, and responsive UI design.



Features

- ✅ Multi-step form navigation
- ✅ Next & Previous step functionality
- ✅ Form validation
- ✅ Responsive design
- ✅ Modern UI using Tailwind CSS
- ✅ State management with React Hooks
- ✅ Easy to customize


 Technologies Used

- React.js
- Tailwind CSS
- JavaScript (ES6)
- Vite
