GitHub User Search Web App

A simple and responsive web application that allows users to search for any GitHub profile and view detailed information along with their latest repositories using the GitHub API.

 Features
 Search GitHub users by username
Display user profile information:
Profile picture
Name
Bio
join date


Public repositories
 Show latest repositories of the user
 Display repository creation date
 Sort repositories (newest to oldest)
 Error handling for invalid usernames
  Modern and responsive UI
Technologies Used


HTML5 – Structure of the web page
CSS3 – Styling and layout
JavaScript (ES6) – Functionality and API handling
GitHub API – Fetching user data


This project uses the official GitHub REST API:

https://api.github.com/users/{username}
https://api.github.com/users/{username}/repos
